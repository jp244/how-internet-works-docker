Wires, Cables, & WiFi

Data are made up of bits, essentially an on or off signal, 0 or 1. Bits are the most basic unit of data and are usually grouped into bytes (8 bits) which are scaled by orders of magnitude to create kilobytes, megabytes, gigabytes, etc. No matter the data type (video, audio, picture) it is sent around the internet and represented as some collection of bits. We need to machines to act as mediums for sending this data since systems relying on complete human translation (e.g. morse code) are inherently error prone. The main foundations in which machines send and receive data are some form of electricity, light, or radio waves.

Terms:

Bandwidth: transmission capacity of a device, measured by bitrate

bitrate: number of bits per second that a system can transmit

latency: amount of time it takes for bit to travel from sender to receiver

ethernet: cheap, can get signal loss and interference over large distances (a few hundred feet) copper cable

need alternative to send bits longer distances to have a global connected network, the solution is to use light, since it moves a lot faster than electricity.

Fiber: very fast, the signal does not degrade over long distances, a lot more expensive than ethernet and hard to work with
